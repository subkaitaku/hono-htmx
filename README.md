# Todo Example

Stack:

* Hono
* JSX (Hono middleware)
* htmx
* Zod
* Cloudflare Workers
* Cloudflare D1

## Usage

Install:

```
npm install
```

Setup:

```
wrangler d1 create todo
wrangler d1 execute todo --local --file=todo.sql
```

PrepareDB:
```
npm run migrateDev
npm run migrateProd
```

Dev:

```
npm run dev
```

Deploy:

```
npm run deploy
```
